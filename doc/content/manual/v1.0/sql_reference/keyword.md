예약어
======

아래는 Rye 키워드(keywords) 중 명령어, 함수명, 타입명 등으로 예약되어 있는 예약어(reserved words)를 정리한 표이다. 사용자는 테이블 이름, 칼럼 이름, 변수 이름과 같은 식별자(identifier)로 아래에 정리된 예약어를 사용할 수 없다. 단, 큰따옴표(" ")나 대괄호(\[ \]) 또는 백틱 부호(\` \`)로 감싸는 방법으로 예약어를 식별자로 사용할 수 있다.
